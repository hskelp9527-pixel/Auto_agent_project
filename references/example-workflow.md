### 生成 PRD.md
```markdown
# Vue 3 电商项目 - Product Requirements Document

## 项目概述
- 项目目标：B2C电商平台
- 目标用户：消费者
- 核心价值：简洁易用的购物体验

## 技术栈
- 前端：Vue 3 + TypeScript
- 构建：Vite
- 样式：Tailwind CSS
- 状态：Pinia
- 路由：Vue Router

## 数据模型
interface Product {
  id: string
  name: string
  price: number
  images: string[]
  category: string
}

interface CartItem {
  productId: string
  quantity: number
}

interface Order {
  id: string
  items: CartItem[]
  totalAmount: number
  status: string
}

## 页面结构
- / - 首页
- /products - 产品列表
- /product/[id] - 产品详情
- /cart - 购物车
- /checkout - 结账
- /orders - 订单列表
- /login - 登录
- /register - 注册

## 验收标准
- 功能完整性：60分
- 设计系统：20分
- 代码质量：10分
- 功能验证：10分
- 目标：≥95分
```

## 关键要点

### 1. 文件驱动
- 所有信息都在文件里
- 换个 AI 也能继续

### 2. 小步快跑
- 每个任务小而独立
- 频繁评估

### 3. 迭代改进
- 加分确认
- 减分重写
- 持续到95分

### 4. 记录一切
- progress.md 记录每次变化
- evaluation.md 记录每次评估
- ERROR_LOG.md 记录每个错误

---

这个示例展示了完整的开发流程。实际项目中可能有更多任务和更多迭代，但核心原则不变。
